# WAVE_BACKEND
Node.js Backend Server for Wave Wallet.

## Install
    $ npm install

## Configure app
Configer Google Drive is located at `gdriveapi-example.json`
- project_id;
- private_key_id;
- private_key = "-----BEGIN PRIVATE KEY-----\n=\n-----END PRIVATE KEY-----\n";
- client_email;
- client_id;
- client_x509_cert_url;

## Running the project
    $ npm start
